## wagmi and viem

**wagmi** is a collection of React Hooks for Ethereum, built on top of **viem**. viem is a low-level TypeScript interface for Ethereum that replaces ethers.js with better performance and type safety.

- wagmi docs v3: https://wagmi.sh/react/getting-started
- wagmi docs v2: https://2.x.wagmi.sh/react/getting-started
- wagmi github repo: https://github.com/wevm/wagmi
- viem docs: https://viem.sh/docs/getting-started
- viem github repo: https://github.com/wevm/viem

## Rainbowkit - Ethereum wallet integration

RainbowKit is a React library that provides a beautiful, customizable wallet connection modal. It's built on top of wagmi and provides an excellent UX out of the box.

- docs: https://www.rainbowkit.com/docs

## shadcn 

shadcn/ui is not a component library you install - it's a collection of re-usable components you copy into your project. Built on Radix UI primitives with Tailwind CSS styling.

- docs: https://ui.shadcn.com
- components: https://ui.shadcn.com/docs/components
- Astro Guide: https://ui.shadcn.com/docs/installation/astro

## visx

visx (formerly vx) is a collection of low-level visualization primitives for React, created by Airbnb. It provides the building blocks to create custom, performant visualizations.

- Visualizations
- docs: https://airbnb.io/visx
- Examples: https://visx.airbnb.tech/gallery
- GitHub: https://github.com/airbnb/visx

## Recharts

Recharts is a composable charting library built on React components. It's higher-level than visx and provides ready-to-use chart components.

- Charts library
- docs: https://recharts.org
- API Reference: https://recharts.github.io/en-US/api/
- Examples: https://recharts.github.io/en-US/examples/
- GitHub: https://github.com/recharts/recharts

## Refine CORE

Refine is a React meta-framework for building data-intensive applications like admin panels, dashboards, and internal tools. It provides hooks and components for CRUD operations, authentication, access control, and more.

- Data Framework for Supabase
- docs: https://refine.dev/core/docs/
- github repo: https://github.com/refinedev/refine
- Blog: https://refine.dev/blog/
